═══════════════════════════════════════════════════════════════════════════════
 MONITORING & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

COMMON ISSUES:
─────────────

Issue: Port already in use
  Fix: Render automatically assigns PORT from environment variable

Issue: Database connection refused
  Fix: Verify DATABASE_URL format: postgres://user:pass@host:5432/db
  Fix: Check SSL setting in src/config/database.js

Issue: JWT authentication fails
  Fix: Ensure JWT_SECRET environment variable is set
  Fix: Verify Authorization header: "Bearer {token}"

Issue: CORS errors from frontend
  Fix: Update FRONTEND_URL in environment variables
  Fix: Add frontend domain to CORS whitelist in server.js

MONITORING:
──────────

• View logs: Render Dashboard → Web Service → Logs
• Check health: GET https://cltech-api.onrender.com/health
• Database monitoring: Render Dashboard → PostgreSQL → Logs
• Response time: Render Dashboard → Metrics

 FRONTEND INTEGRATION

Update your React frontend configuration:

FILE: frontend/.env.production

VITE_API_URL=https://cltech-api.onrender.com

FILE: frontend/src/config/api.js (or similar)
──────────────────────────────────────────────
const API_URL = import.meta.env.VITE_API_URL || 'https://cltech-api.onrender.com';

Example API call:
─────────────────
import axios from 'axios';

const api = axios.create({
  baseURL: process.env.VITE_API_URL
});

api.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

export default api;
