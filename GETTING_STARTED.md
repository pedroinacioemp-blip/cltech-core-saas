# 🚀 GETTING STARTED - CL TECH CORE

**Bem-vindo ao CL TECH CORE!**

Este é um guia ultra-rápido para você começar em menos de 5 minutos.

---

## ⚡ Setup Ultra-Rápido (5 minutos)

### 1️⃣ Clone/Abra o Projeto
```bash
cd C:\Users\SEU_USUARIO\Desktop\CL-TECH-CORE
```

### 2️⃣ Instale Dependências
```bash
npm install
```
*Aguarde 2-3 minutos...*

### 3️⃣ Inicie a Aplicação
```bash
npm start
```

**Pronto! 🎉 A aplicação abrirá em segundos!**

---

## 🎮 Primeiros Passos

### Compile Seu Primeiro C++

1. **Cole este código no editor:**
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, CL TECH CORE!" << endl;
    return 0;
}
```

2. **Pressione `Ctrl+Enter`** ou clique **RUN**

3. **Veja o resultado no console!** 🎊

---

### Compile Seu Primeiro Java

1. **Cole este código no editor:**
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, CL TECH CORE!");
    }
}
```

2. **Pressione `Ctrl+Enter`** ou clique **RUN**

3. **Veja o resultado no console!** 🎊
